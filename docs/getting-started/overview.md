Eagle 🦅一套轻量级 Go 微服务框架

### Goals

🦅一款基于Go构建的微服务框架，可以快速构建API服务进行业务开发，遵循SOLID设计原则


### Features

* Config：支持多种数据格式及热加载；
* Transport：通用的 HTTP/gRPC 传输层；
* Logger：标准日志接口，可方便集成三方 log 库；
* Metrics：统一指标接口，可以实现各种指标系统，默认集成 Prometheus；
* Tracing：支持微服务链路追踪，遵循 OpenTelemetry 规范定义；
* Errors：统一返回错误码格式；
